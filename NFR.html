Non Functional Requirements

Performance requirements

1) The number of online users of the HRMS can be estimated as 50 at most. [1]

2) There is no restriction on the number of the users to be added to the database. [1]

Hardware Requirements

1) The HRMS application will be storing employees’ personal data of roughly 1000 employees. Roughly 2 GB of storage capacity is needed. [1]

Software Requirements

1) Internet connection must be established since HRMS application will be a web-based application,.

2) The HRMS software will be used on PCs and will function via internet or interanet in any web browser.

3) The HRMS software will interact with Account Management System with SOAP or RESTful Web Services.

4) The HRMS application interface will be developed in Java (J2EE) frameworks.

5) The HRMS software personal database model will support MYSQL environment as DBMS.

6) The HRMS software will support JDK environment. The HRMS will run on any platform supporting JDK technology.

7) Application will run on 256MB or higher of RAM.

Labour Laws

All employees have basic rights in the workplace, including the right to privacy, fair compensation, and freedom from discrimination. The organization abides by the Equal Employment Opportunity and the Americans with Disabilities Act laws. Usability

The user should have basic knowledge to operate web applications.
